Entity Reconciliation CG charter
================================

This hosts the charter of our community group, which can be read at:
https://reconciliation-api.github.io/charter/

## Development

To view branch and development versions of the WG charter locally:

* Clone the repo: `$ git clone git@github.com:reconciliation-api/charter.git`
* Change into the directory: `$ cd charter`
* Check out the respective branch with `$ git switch $branch` and/or carry out local changes.
* Open the WG charter draft: http://127.0.0.1:8000/working_group_charter.html
